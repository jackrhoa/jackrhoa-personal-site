# Data files

- `sports.ts` — sport abbreviation → full name mapping
- `networks.ts` — network name → logo URL + scale factor
- `positions.ts` — position abbreviation → full name (e.g. TD → TECHNICAL DIRECTOR)
- `teams.ts` — auto-generated by `scripts/fetchTeams.js`, do not edit manually. Re-run the script to update.

## Schedule calendar event format

```
(CANCELED) POSITION (HOOVISION) - SPORT vs AWAY_TEAM (NETWORK)
```

- `(CANCELED)` — optional, prepended to position field; renders a red X over the date/time
- `(HOOVISION)` — optional, in the position field; replaces the network slot with "HOOVISION" text
- `SPORT` — accepts abbreviation (`CFB`) or full name (`FOOTBALL`); always displayed as abbreviation
- `AWAY_TEAM` — multi-word names supported (e.g. `VIRGINIA TECH`)
- `(NETWORK)` — optional; if unrecognized, displays as raw text badge
- Home team is always UVA
- Must contain ` vs ` to render as a game card; anything else renders as a raw event row
